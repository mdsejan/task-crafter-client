# TaskCrafter - Efficient Task Management

This project is a task management platform that simplifies organizing, prioritizing, and tracking tasks, catering to diverse professional needs.

TaskCrafter - Server Side Repo [click Here](https://github.com/mdsejan/task-crafter-server)

---

## Project Features

### User Authentication

- Register, log in, and log out functionalities
- Google sign-in and multi-login system implemented

### Task Management Dashboard

- Create, edit, and delete tasks
- Task details include titles, descriptions, deadlines, and priority levels (low, moderate, high)
- Categorization of tasks into to-do, ongoing, and completed lists
- Drag-and-drop functionality for task movement across lists

### Additional Features

- Toast notifications for task assignments, updates, and deadlines
- Utilization of React Hook Form for task creation
- Task editing functionality enabling users to modify task details
